Welcome! This is a basic document on how to install The files properly to Eciplse. Needs Prereqs

Prereqs:
  • Windows (Mac fix is on the way.)
  • Eclipse Mars
  • Git Extention
Install Instructions
  1. open Eclipse Mars
  2. Begin Importing
  3. Got to File, import
  4. Git
    • Projects from Git.
  5. Clone URI
  6. Source Git Repository
    • Location Settings
      • URI: https://github.com/soft-eng-practicum/SpaceInvaders
      • Host: github.com
      • Repository path: /soft-eng-practicum/SpaceInvaders
    • Connection Settings
      • Protocul https
      • Port: null (Leave it blank)
    • Authentication Setting
      • User: YOUR_USER_NAME
      • Password: YOUR_PASSWORD:
  7. Select Master
  8. Choose your preferred Save Location
  9. Import as General Project

If you find any issues with this, please let me know.
